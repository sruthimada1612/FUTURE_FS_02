# Task 2: Client Lead Management System (Mini CRM)

This repository contains the Full-Stack Mini CRM workspace application designed to ingest and organize client interactions dynamically.

## 🛠️ Tech Stack
- **Frontend:** React.js, Stateful CSS
- **Backend Architecture:** Node.js, Express.js REST API
- **Database Engine:** MySQL Relational Database

## 🚀 Key Features
- **Pipeline Dashboards:** Renders visual status metric counters (New, Contacted, Converted) that filter the main grid data.
- **Interaction Ledger:** A built-in modal interface to save time-stamped follow-up notes for specific leads.
- **Relational Integrity:** Backed by a clean 1:M (One-to-Many) database structure linking leads to historical log entries.
